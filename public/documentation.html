API myFlix

This application provides access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Request URL Response
GET /users Returns a list of all users
POST /users Creates a new user
PUT /users/:id Updates a user's information
POST /users/:id/:filmTitle Adds a movie to a user's favorite list
DELETE /users/:id/:filmTitle Removes a movie from a user's favorite list
DELETE /users/:id Deletes a user
GET /films Returns a list of all movies
GET /films/:title Returns information about a specific movie
GET /films/genre/:genreName Returns information about a specific genre
GET /films/directors/:directorName Returns information about a specific director
