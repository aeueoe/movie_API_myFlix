API myFlix

This application provides access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Request URL Response Description
GET /users Returns a list of all users N/A
POST /users Creates a new user {"Username": "", "Email": "", "Password": "", "Birthday": ""}
PUT /users/:Username Updates a user's information {"Username": "", "Email": "", "Password": "", "Birthday": ""}
POST /users/:Username/favoriteMovies/:movieId Adds a movie to a user's favorite list {"movieId": ""}
DELETE /users/:Username/favoriteMovies/:movieId Removes a movie from a user's favorite list N/A
DELETE /users/:Username Deletes a user N/A
GET /movies Returns a list of all movies N/A
GET /movies/:title Returns information about a specific movie N/A
GET /movies/genres/:genreName Returns information about a specific genre N/A
GET /directors Returns a list of all directors N/A
GET /directors/:directorName Returns information about a specific director N/A
GET /actors Returns a list of all actors N/A
GET /actors/:actorName Returns information about a specific actor N/A
